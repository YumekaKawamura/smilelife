# smilelfe
PHP自作（学校教養システム）  

## 作成した背景
2020年から義務教育でプログラミングが必修化され、内容は基本的に自由ですが２つの学習活動が定義されています。  
1つ目はタイピング等の基本的な活用スキルの習得、2つ目はプログラミング的な論理的思考を習得するための活動です。  
上記の背景を踏まえて、毎日のタイピング練習や身近にコンピュータがある環境を生み出すためにこのシステムを開発しました。
